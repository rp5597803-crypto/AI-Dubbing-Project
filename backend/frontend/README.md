# AI Video Dubbing Studio

An AI-powered video dubbing project built with Python, Flask, FFmpeg, Whisper, Translation and Text-to-Speech.

## Features

- Upload MP4 video
- Extract audio using FFmpeg
- Speech-to-Text using Whisper
- AI Translation
- Text-to-Speech
- Replace original audio
- Download dubbed video
